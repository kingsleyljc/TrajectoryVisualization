
# TrajectoryVisualization轨迹点可视化与坐标系转换
Visualize trajectory points in .csv based on Baidu Map.
> 填写本地相对目录点击获取按钮，即可显示该目录下的所有csv文件
> 选择csv文件的原坐标系，即可自动转换为baidu坐标系并显示在百度地图中。
> 坐标系转换支持gcj、wgs、bd坐标系之间的转换
